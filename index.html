Learning HTML - The Basics

HTML consists of the following:

  • An opening tag
  • A closing tag
  • Some content

These three components make up an HTML element.

Learning CSS - The Basics

CSS (Cascading Style Sheets) is the code that styles web content.

CSS consists of the following:

  • A selector
  • A property
  • A property value
  • A declartation

There four components make up a CSS ruleset.
